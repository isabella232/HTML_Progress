Introduction

This example will run a default ProgressBar Monitor, and used a default QuickForm renderer without any form template customizations. User callback 'logger' will write into a file through PEAR::Log.

[Top]

 PHP script

User-Callback


<?php
// progressHandler.php

function logger($progressValue, &$obj)
{
    include_once 'Log.php';
    $logger = &Log::singleton('file', 'monitor.log', $_SERVER['REMOTE_ADDR']);

    if (fmod($progressValue,25) == 0) {
        $logger->info("$progressValue % has been reached");
    } else {
        $logger->debug("Progress ... $progressValue %");
    }
}

?>
