Introduction

This example requires :

  • PEAR::HTML_Progress_Monitor 1.2.0RC3 or better (see source at line 11).

Be aware that:

  • HTML_Progress_Monitor has dependencies (HTML_CSS 0.3.4+, HTML_QuickForm 3.1.1+)
  • This example requires PEAR::Log (1.8.0+)

This example will logs a default ProgressBar Monitor, and used a default QuickForm renderer without any form template customizations.

User callback 'logger' (defined on lines 13 to 23) which write into a file through PEAR::Log was attached on progress bar at line 26.
