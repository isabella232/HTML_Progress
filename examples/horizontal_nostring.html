Introduction

This example will run a basic horizontal ProgressBar, but without percent text info.

There are 10 cells with default values:

  • active-color = #006600
  • inactive-color = #CCCCCC
  • cell width = 15px
  • cell height = 20px
  • cell spacing = 2px
Percent text info is hidden.

[Top]

 PHP script

Build the progress bar

require_once 'HTML/Progress.php';

$bar = new HTML_Progress();
$bar->setAnimSpeed(100);
$bar->setIncrement(10);
$bar->setStringPainted(true);   // get space for the string
$bar->setString('');            // but don't paint it

Loop to run the progress


do {
    $bar->display();
    if ($bar->getPercentComplete() == 1) {
        break;   // the progress bar has reached 100%
    }
    // your user-process should be put HERE !
    $bar->incValue();
} while(1);
